get /users 获取所有的用户信息

查询参数
名称类型描述
page int 显示第几页的内容
size int 每页显示的数量

SUCCESS:200, ok

参数:
名称类型描述
count int 符合条件的所有用户数量
users array 用户列表。
示例:
{
    "count": 500,
    "users": [
        {"id":1, "username": "admin1", "name": "管理员2"},
        {"id":2, "username": "admin2", "name": "管理员2"}
    ],
}
<users count="500">
    <user id="1" username="admin1" name="管理员1" />
    <user id="2" username="admin2" name="管理员2" />
</users>
